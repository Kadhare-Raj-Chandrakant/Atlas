# Atlas — Vision

## Mission

Atlas is a local-first desktop application that automatically builds a connected memory graph from natural writing. The user writes naturally. Atlas remembers and connects everything else.

## Core Philosophy

Traditional note-taking forces users into organizational overhead — folders, tags, hierarchies, manual linking. Atlas eliminates this friction. Every piece of text the user writes is automatically understood, connected, and surfaced at the moment it becomes relevant.

The user should never need to organize. They only need to write.

## Goals

- Provide a distraction-free writing environment
- Automatically extract entities, concepts, and relationships from text
- Build a persistent graph of connected knowledge over time
- Surface relevant past context when writing
- Enable full-text and semantic search across all content
- Operate entirely locally with no cloud dependency
- Remain fast and responsive with years of usage

## Non-Goals

- Atlas is not a replacement for structured databases or spreadsheets
- Atlas is not a project management tool
- Atlas is not a social or collaborative platform
- Atlas is not a publishing platform

## Guiding Principles

1. **The user writes naturally. Atlas remembers everything else.**
2. Local-first and private by default.
3. Zero overhead for the user — no tags, no folders, no manual linking.
4. The graph emerges from writing, not the other way around.
5. Performance is a feature. Every interaction must feel instant.
6. Ship a clear, focused product rather than a feature-covered monolith.
7. Prefer boring, reliable technology over novelty.
